# Quipu Pacha

*Recording your spacetime*

Track time across the digital world. All data stored locally.

**Pacha** - spacetime (Quechua)
**Quipu** - Inca knot-based data recording

## How It Works

- Only tracks when tab is **active and focused**
- Pauses on window switch or idle (60s)
- Saves every 30 seconds
- Groups by domain

## Privacy

- All data in local storage - never leaves your device
- No servers, no accounts, no analytics
- Export or delete anytime

## Install

1. Clone repo
2. Chrome - chrome://extensions/
3. Enable **Developer mode**
4. **Load unpacked** - select folder

## Permissions

| Permission | Why |
|------------|-----|
| tabs | Detect active tab |
| storage | Save locally |
| idle | Pause when away |
| favicon | Site icons |
